# Programas de ejemplo

Ejemplos canónicos mínimos de programas escritos en el lenguaje del SV.

## Estado

Activo con dos piezas mínimas verificables.

## Ejemplos publicados

| Archivo | Qué muestra | Estatuto |
|---------|-------------|----------|
| `celula_basica_con_evaluacion.svp` | célula mínima con `cellstate` y `evaluate` | ejemplo canónico básico |
| `consulta_framecomparison.svp` | consulta con `query_spec` de tipo `FrameComparison` y contexto `FrameComparison(F1, F2)` | ejemplo canónico de verificabilidad superficial |

## Alcance

Estos ejemplos no pretenden cubrir todavía toda la expresividad del lenguaje ni sustituir la suite de conformidad. Su función es ofrecer una superficie pública mínima y ejecutable para terceros.

Las piezas diseñadas como sondas de contraste adversarial se alojan aparte en `tests/adversarial/documentados/`, para no confundir ejemplos canónicos con materiales de tensión técnica.
