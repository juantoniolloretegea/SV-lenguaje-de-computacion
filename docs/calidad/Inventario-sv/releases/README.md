# Releases del inventario SV

## V.1

Fecha de arranque: 2026-06-24.

La versión V.1 fija la estructura del inventario, la publicación matriz en Markdown, el CSV de mantenimiento, el esquema de columnas y los criterios de registro.

## Regla de nueva release

Procede nueva release cuando se incorpore un lote relevante de publicaciones, se modifique el esquema de columnas, se depuren duplicados de forma significativa, se cambie el DOI de citación preferente de una publicación o se sincronice una versión pública en PubPub.
