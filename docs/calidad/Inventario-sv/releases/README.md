# Releases del inventario

Este directorio conserva notas de actualización del inventario.

## Regla de release

Una nueva release procede cuando el inventario incorpora nuevas publicaciones, corrige DOI, modifica estados editoriales o reorganiza relaciones entre colecciones.

## Flujo recomendado

1. Actualizar `data/inventario_publicaciones_sv.csv`.
2. Actualizar la tabla de `inventario_publicaciones_sv.md`.
3. Revisar que no haya duplicados de publicaciones canónicas.
4. Subir a GitHub.
5. Copiar la tabla renderizada o el contenido completo hacia PubPub.
6. Publicar o actualizar la release en PubPub.
