# Biblioteca estándar

Implementaciones nativas de los objetos y operadores del Sistema Vectorial SV.

**Estado:** Pendiente (Fase II).

**Módulos previstos:**
- `tri` — tipo ternario primitivo
- `cell` — célula SV(n, b)
- `classify` — motor de clasificación T(n)
- `gamma` — función de criticidad Γ
- `compose` — operadores de composición (σ, ⊗_gate, ▷, Comp)
- `connector` — conectores tipados φ
- `frame` — evaluación inmutable
- `trajectory` — trayectoria append-only
- `query` — consulta auditable
