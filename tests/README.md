# Suite de conformidad

Tests que verifican que el lenguaje preserva los invariantes del Sistema Vectorial SV.

**Estado:** Pendiente (Fase IV).

**Categorías previstas:**
- Conformidad de tipos (Tri no degrada a entero/booleano/null)
- Conformidad estructural (n = b², b ≥ 3)
- Conformidad de clasificación (T(n) = ⌊7n/9⌋)
- Conformidad de composición (operadores lícitos, R obligatoria)
- Conformidad de auditoría (inmutabilidad de frames, trazabilidad)
- Paridad cruzada (Rust = Python = WASM para toda entrada)
